# Retrieval Kit

A small library for document ingestion, vector and keyword search to aid the creation of RAG systems and MCP servers for knowledge retrieval.

## Status

This repository is currently an early scaffold. The public API exists as method stubs in [`src/lib.rs`](./src/lib.rs).

Planned capabilities include:

- vector embeddings
- single-document and batch chunking and ingestion
- vector and keyword search
- document listing, lookup, and deletion
- tool definitions and invocation for MCP-style knowledge retrieval integrations
